# Column Types Reference

Full table of Dataiku column types and their Python equivalents.

## Common Column Types

| Dataiku Type | Description | Python Equivalent |
|--------------|-------------|-------------------|
| `string` | Text | str |
| `int` | Integer | int |
| `bigint` | Large integer | int |
| `double` | Decimal | float |
| `float` | Decimal (smaller) | float |
| `boolean` | True/False | bool |
| `date` | Date only | datetime.date |
| `object` | Complex/JSON | dict |
